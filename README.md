# 발주모아 파트너스

발주모아 파트너스 애플리케이션입니다.

## 기능

- 카카오 브랜드메시지 목록 화면
  - 메시지 유형 필터링 (전체/신상품/이벤트/가격할인)
  - 기간 및 발송채널 필터
  - 탭별 보기 (발송 예정 / 발송 완료 / 임시저장)
  - 메시지 카드 표시
  - 페이지네이션
  - 빈 상태 UI

## 시작하기

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```

3. 브라우저에서 [http://localhost:3000](http://localhost:3000) 열기.

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
